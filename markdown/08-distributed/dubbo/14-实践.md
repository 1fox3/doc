### Arthas

#### Alibaba开源的java诊断工具

#### git地址

- https://github.com/alibaba/arthas

#### 安装

- wget https://alibaba.github.io/arthas/arthas-boot.jar

#### 使用

- 启动
  - java -jar arthas-boot.jar

- 查看源码
  - jad org.apache.dubbo.rpc.Protocol$Adaptive

- 查看引用
  - trace org.apache.dubbo.rpc.Filter *
