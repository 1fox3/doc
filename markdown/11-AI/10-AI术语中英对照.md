## 面试常用英文表达

1. We used RAG to ground the model with internal documents.
2. The model is not treated as a source of truth; it generates answers based on retrieved context.
3. Permission filtering is enforced before retrieval, not by the model itself.
4. We use schema validation to make structured outputs reliable.
5. For high-risk actions, the agent requires human confirmation.
6. We monitor latency, token usage, cost, and answer quality.
7. We use a smaller model for simple tasks and a stronger model for complex reasoning.
8. Prompt injection is handled through defense in depth, not just prompt rules.
