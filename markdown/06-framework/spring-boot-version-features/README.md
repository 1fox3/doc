# Spring Boot 版本升级

本目录只保留 Spring Boot 版本演进和项目升级相关的技术内容。重点关注运行时要求、配置加载、自动配置、Actuator、观测性、Jakarta 迁移和第三方依赖兼容。

## 版本主线

- Boot 1.x 到 2.x：基础框架升级到 Spring 5，WebFlux、Actuator 2、Micrometer 和配置属性绑定模型变化。
- Boot 2.4 到 2.7：ConfigData、路径匹配策略、循环依赖默认策略、Java 版本基线和依赖生态调整。
- Boot 3.x：Java 17、Spring Framework 6、Jakarta EE 9、AOT、Native Image、Micrometer Observation。
