# Nacos

Nacos 同时提供服务发现、健康检查、动态配置、动态 DNS、服务元数据管理和权限控制。它既可以作为 Spring Cloud Alibaba 的注册配置中心，也可以通过 OpenAPI、SDK 或 DNS 方式被其他系统接入。

## 核心关注点

- 数据模型：Namespace、Group、Service、Cluster、Instance、DataId。
- 注册发现：临时实例、持久实例、心跳、健康检查、上下线状态和元数据。
- 配置治理：动态刷新、版本、灰度、回滚、权限、加密和本地缓存。
- 通信机制：Nacos 2.x gRPC 长连接和端口偏移。
- 一致性：Distro 最终一致性和 Raft/JRaft 强一致性。
