# 微服务

本目录按技术体系重新划分为两条主线：Spring Cloud 和 Nacos。仅保留微服务工程落地相关内容，删除脑图模板、复习话术和非技术占位内容。

## 目录

- [Spring Cloud](01-spring-cloud/README.md)：微服务架构、服务治理、调用网关、弹性可观测、安全、Spring Cloud Alibaba 生态。
- [Nacos](02-nacos/README.md)：注册发现、配置中心、数据模型、通信机制、一致性、部署和安全治理。

## 使用建议

- 先看 Spring Cloud 主线，建立微服务治理全貌。
- 再看 Nacos 主线，掌握注册中心和配置中心的关键实现细节。
- 生产落地时同时关注超时、重试、熔断、限流、灰度、动态配置、链路追踪和分布式一致性。
