# 分布式集群与高可用

## 节点角色

现代 Elasticsearch 使用 `node.roles` 配置节点角色。

- master-eligible：参与主节点选举，管理集群状态。
- data：保存分片并执行读写，可细分 hot、warm、cold、frozen、content 等层级。
- ingest：执行 ingest pipeline。
- coordinating only：不保存数据、不参与主节点选举，只做请求分发和结果归并。
- ml、transform、remote_cluster_client：用于机器学习、转换任务和跨集群访问。

生产集群通常使用 3 个专用 master-eligible 节点，数据节点按冷热层或业务负载拆分。

## 主节点与集群状态

主节点负责维护 cluster state，包括索引元数据、mapping、分片分配和节点信息。主节点不负责所有查询，也不应承担重数据查询压力。

风险点：

- mapping 过大或字段爆炸会导致 cluster state 膨胀。
- 节点频繁上下线会导致主节点忙于分片分配。
- 专用 master 节点资源过小会影响整个集群稳定性。

## 发现与选举

7.x 之后集群协调机制替代 Zen Discovery。常见配置包括：

- `cluster.name`：集群名称。
- `node.name`：节点名称。
- `network.host`：绑定地址。
- `discovery.seed_hosts`：种子节点。
- `cluster.initial_master_nodes`：新集群首次启动的候选主节点列表。

`cluster.initial_master_nodes` 只用于新集群 bootstrap，集群形成后不应长期保留错误配置。

## 分片与副本

primary shard 决定索引的水平拆分能力，replica shard 提供查询扩展和高可用。

规划原则：

- 单分片过大恢复慢，过小会产生管理开销。
- 常见单分片目标大小可从 20 GB 到 50 GB 评估，日志场景也要结合保留周期和查询模式。
- 主分片数量创建后不能直接增加，可通过 split、reindex 或 rollover 调整。
- 副本数可动态调整，但会增加写入复制成本。

## 分片分配

分片分配受节点角色、磁盘水位、allocation awareness、过滤规则和 rebalance 策略影响。

常用配置：

- `cluster.routing.allocation.awareness.attributes`：按机架、可用区分散副本。
- `cluster.routing.allocation.same_shard.host`：避免同一主副分片在同一物理机。
- `cluster.routing.allocation.disk.watermark.low`：磁盘超过低水位后不再分配新分片。
- `cluster.routing.allocation.disk.watermark.high`：超过高水位后尝试迁移分片。
- `cluster.routing.allocation.disk.watermark.flood_stage`：超过洪泛水位后索引可能被置为只读保护。

## 集群健康

- green：所有主分片和副本分片都已分配。
- yellow：主分片已分配，但至少一个副本未分配。
- red：至少一个主分片未分配，部分数据不可用。

排查 red/yellow 重点看 `_cluster/allocation/explain`、`_cat/shards`、磁盘水位、节点角色、索引 allocation 过滤和副本数。

## 搜索分布式执行

客户端请求到任意节点，该节点成为协调节点。协调节点向相关分片发起 query 阶段请求，归并后再进入 fetch 阶段拉取文档内容。

协调节点需要足够 CPU 和堆内存处理排序、聚合和结果归并。大型聚合或深分页会让协调节点成为瓶颈。

## 跨集群能力

Tribe node 已废弃。现代方案包括：

- Cross Cluster Search：跨集群查询。
- Cross Cluster Replication：跨集群复制，需要相应许可和版本兼容。
- Snapshot/Restore：跨集群迁移和灾备恢复。

跨集群查询要关注网络延迟、远端集群可用性、索引命名冲突和权限模型。

## 高可用建议

- 至少 3 个 master-eligible 节点，避免脑裂和单点。
- 主副分片跨机器、跨机架或跨可用区分布。
- 保持磁盘水位安全，预留 merge、恢复和快照空间。
- 避免一个集群承载过多不相关业务，必要时按业务或负载拆集群。
- 定期演练节点故障、分片恢复和快照恢复。
