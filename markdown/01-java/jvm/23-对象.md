### 对象创建

#### 类加载检查

- 遇到new指令时，首先定位检查常量池中类的符号引用，并检查类是否已被加载、解析和初始化过，如果没有则进行相应的类加载过程

#### 分配内存

- 为对象分配内存空间
  - 方式
    - 指针碰撞
    - 空闲列表
  - 内存分配并发问题
    - CAS+失败重试
    - TLAB，为每一个线程预先在Eden分配一块内存

#### 初始化零值

- 内存分配后，虚拟机需要将分配的内存空间都初始化为0值，不包括对象头，以保证不符初始值就可以直接使用，程序能访问到字段的数据类型所对应的零值
