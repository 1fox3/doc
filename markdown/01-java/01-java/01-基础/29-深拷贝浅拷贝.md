# 深拷贝浅拷贝

## 核心区别

| | 浅拷贝 | 深拷贝 |
|-|--------|--------|
| 基本类型字段 | 复制值 | 复制值 |
| 引用类型字段 | 复制引用地址，新旧对象共享内部对象 | 递归复制，新旧对象完全独立 |

## 实现方式

| 方式 | 说明 |
|------|------|
| 拷贝构造器 / 转换方法 | 手动实现，最清晰可控，推荐 |
| `clone()`（实现 `Cloneable`） | 默认浅拷贝，深拷贝需手动递归处理引用字段 |
| 序列化再反序列化 | 可实现深拷贝，但性能较差，需实现 `Serializable` |
| MapStruct / BeanUtils 等工具 | 默认浅层复制，使用时需明确其拷贝深度 |
