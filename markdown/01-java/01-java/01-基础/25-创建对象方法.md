# 创建对象方法

## 常见方式

| 方式 | 示例 | 说明 |
|------|------|------|
| `new` 关键字 | `new DemoClass()` | 最常用，直接调用构造方法 |
| 反射 `newInstance`（已废弃） | `DemoClass.class.newInstance()` | 只能调用无参构造，Java 9+ 已废弃 |
| 反射 `getConstructor` | `DemoClass.class.getConstructor().newInstance()` | 可指定构造器参数，推荐替代上一种 |
| `Class.forName` + `newInstance` | `Class.forName("com.example.DemoClass").newInstance()` | 动态加载类，常用于插件、SPI 场景 |
| `clone` | 实现 `Cloneable` 后调用 `clone()` | 默认浅拷贝，语义复杂，不推荐 |
| 反序列化 | `ObjectInputStream#readObject()` | 绕过构造器，恢复对象状态 |
| 工厂方法 / Builder / 容器 | `SpringContext.getBean()`、`XxxBuilder.build()` | 业务代码推荐方式 |
