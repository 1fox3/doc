# 包装类

## 基本类型 vs 包装类

| 维度 | 基本类型 | 包装类 |
|------|----------|--------|
| 用途 | 常量、局部变量 | 泛型、集合、反射、空值表达 |
| 存储位置 | 局部变量在栈；成员变量在堆 | 对象，几乎都在堆 |
| 占用空间 | 较小 | 较大 |
| 默认值 | 非 null（如 `0`、`false`） | `null` |
| 比较方式 | `==` 比较值 | `==` 比较内存地址；值比较用 `equals` 或转为基本类型 |

## 自动装箱与拆箱

- 装箱：基本类型 → 包装类，底层调用 `valueOf()`。
- 拆箱：包装类 → 基本类型，底层调用 `xxxValue()`（如 `intValue()`）。
- 缓存：`valueOf()` 会使用缓存，`Integer` 缓存 `-128~127`，`Byte`/`Short`/`Long`/`Character` 有类似范围。

## 注意点

- 不要用 `==` 比较包装类数值，应使用 `equals` 或先转为基本类型。
- 包装类为 `null` 时自动拆箱会抛 `NullPointerException`，数据库字段、JSON 入参、RPC DTO 中需特别注意。

## 示例

```java
Integer a = 127;
Integer b = 127;
System.out.println(a == b); // true，命中缓存

Integer c = 128;
Integer d = 128;
System.out.println(c == d); // false，超出缓存范围

Integer value = null;
// int n = value; // 自动拆箱抛 NullPointerException
```
