# 变量

## 成员变量 vs 局部变量

| 维度 | 成员变量 | 局部变量 |
|------|----------|----------|
| 定义位置 | 类中、方法外 | 方法、构造器或代码块中 |
| 修饰符 | 可用访问修饰符、`static` 修饰 | 不可用访问修饰符和 `static`，但可用 `final` |
| 存储位置 | 堆内存（`static` 则属于类） | 栈内存 |
| 生命周期 | 随对象创建而存在，GC 后释放 | 随方法调用入栈，方法结束后失效 |
| 默认值 | 自动赋类型默认值（`final` 时必须显式赋值） | 无默认值，使用前必须显式初始化 |

## 静态变量

- 使用 `static` 修饰，属于类本身，所有实例共享，随类加载初始化，Class 卸载时释放。

## 注意点

- `final` 只保证引用不可变，不保证引用对象的内部状态不可变。
