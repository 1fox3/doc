# JDK8提供的内置的函数式接口

## 常用接口总览

| 接口 | 抽象方法签名 | 输入 → 输出 | 典型用途 |
|------|-------------|-------------|----------|
| `Function<T,R>` | `R apply(T t)` | T → R | 转换，`Stream.map` |
| `Consumer<T>` | `void accept(T t)` | T → 无 | 消费，`Stream.forEach` |
| `Supplier<T>` | `T get()` | 无 → T | 生产/懒加载，`Optional.orElseGet` |
| `Predicate<T>` | `boolean test(T t)` | T → boolean | 过滤，`Stream.filter` |
| `UnaryOperator<T>` | `T apply(T t)` | T → T | 同类型转换，`List.replaceAll` |
| `BinaryOperator<T>` | `T apply(T t1, T t2)` | T,T → T | 合并，`Stream.reduce` |
| `BiFunction<T,U,R>` | `R apply(T t, U u)` | T,U → R | 双参数转换 |
| `BiConsumer<T,U>` | `void accept(T t, U u)` | T,U → 无 | 双参数消费，`Map.forEach` |
| `BiPredicate<T,U>` | `boolean test(T t, U u)` | T,U → boolean | 双参数判断 |

## 组合能力

| 接口 | 组合方法 | 说明 |
|------|----------|------|
| `Function` | `andThen` / `compose` | `andThen` 从左到右，`compose` 从右到左 |
| `Predicate` | `and` / `or` / `negate` | 逻辑与、或、非 |
| `Consumer` | `andThen` | 依次执行多个消费操作 |

## Consumer\<T>

```java
// 基础用法
Consumer<String> print = System.out::println;
print.accept("hello"); // hello

// andThen：依次执行多个消费操作
Consumer<String> log = s -> System.out.println("[LOG] " + s);
print.andThen(log).accept("world");
// world
// [LOG] world

// BiConsumer：两个输入，无返回
BiConsumer<String, Integer> showIdx = (name, i) -> System.out.println(i + ": " + name);
showIdx.accept("Alice", 0); // 0: Alice

// Map.forEach 接受 BiConsumer
Map.of("Alice", 90, "Bob", 85).forEach((k, v) -> System.out.println(k + " -> " + v));
```

## Supplier\<T>

```java
// 工厂方法：每次调用创建新实例
Supplier<List<String>> listFactory = ArrayList::new;
System.out.println(listFactory.get() == listFactory.get()); // false

// orElse vs orElseGet：orElseGet 只在 Optional 为空时才调用 Supplier（懒执行）
Optional<String> opt = Optional.empty();
String lazy = opt.orElseGet(() -> expensiveCompute());
```

## Predicate\<T>

```java
Predicate<String> notBlank   = s -> !s.isBlank();
Predicate<String> longEnough = s -> s.length() >= 3;

// and / or / negate 组合
Predicate<String> valid   = notBlank.and(longEnough);
Predicate<String> isBlank = notBlank.negate();

System.out.println(valid.test("hi"));   // false（长度不足）
System.out.println(valid.test("hey"));  // true

// Stream.filter
List.of("a", "hello", "", "world")
    .stream()
    .filter(notBlank.and(longEnough))
    .forEach(System.out::println); // hello, world

// BiPredicate：两个参数
BiPredicate<String, Integer> longerThan = (s, n) -> s.length() > n;
System.out.println(longerThan.test("hello", 3)); // true
```

## Function\<T, R>

```java
Function<String, Integer> strLen = String::length;
Function<Integer, String> intStr = i -> "len=" + i;

// andThen：从左到右串联（先 strLen，再 intStr）
System.out.println(strLen.andThen(intStr).apply("hello")); // len=5

// compose：从右到左串联（先执行参数，再执行 this）
System.out.println(strLen.compose(String::toUpperCase).apply("hello")); // 5

// 多步串联
Function<String, String> process = ((Function<String, String>) String::strip)
    .andThen(String::toLowerCase)
    .andThen(s -> s.replace(" ", "_"));
System.out.println(process.apply("  Hello World  ")); // hello_world

// BiFunction：两个输入，一个输出
BiFunction<String, Integer, String> repeat = (s, n) -> s.repeat(n);
System.out.println(repeat.apply("ha", 3)); // hahaha
```

## UnaryOperator\<T>

```java
UnaryOperator<String> toUpper = String::toUpperCase;
UnaryOperator<String> trim    = String::strip;

// List.replaceAll 接受 UnaryOperator
List<String> list = new ArrayList<>(List.of("  alice  ", "  BOB  ", "charlie"));
list.replaceAll(s -> trim.andThen(toUpper).apply(s));
System.out.println(list); // [ALICE, BOB, CHARLIE]
```

## BinaryOperator\<T>

```java
BinaryOperator<Integer> add  = Integer::sum;
BinaryOperator<String>  join = (a, b) -> a + ", " + b;

System.out.println(add.apply(3, 4));           // 7
System.out.println(join.apply("hi", "there")); // hi, there

// minBy / maxBy
BinaryOperator<String> longer = BinaryOperator.maxBy(Comparator.comparingInt(String::length));
System.out.println(longer.apply("hi", "hello")); // hello

// Stream.reduce 接受 BinaryOperator
int sum = List.of(1, 2, 3, 4, 5).stream().reduce(0, Integer::sum);
System.out.println(sum); // 15

String sentence = List.of("Java", "is", "fun").stream()
    .reduce("", (a, b) -> a.isEmpty() ? b : a + " " + b);
System.out.println(sentence); // Java is fun
```
