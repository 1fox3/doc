# 位移运算符

## 三种运算符

| 运算符 | 名称 | 规则 | 等价操作 |
|--------|------|------|----------|
| `<<` | 左移 | 低位补 0 | 乘以 2 的幂（不溢出时） |
| `>>` | 有符号右移 | 高位补符号位 | 除以 2 的幂（保留符号） |
| `>>>` | 无符号右移 | 高位补 0 | 不考虑符号的右移 |

## 注意点

- **位移越界**：位移位数超过类型位宽时会先取模再位移。`int` 取低 5 位（模 32），`long` 取低 6 位（模 64），因此 `x << 42` 等同于 `x << 10`。
- **常见用途**：状态标记、权限位、哈希扰动、网络协议解析、高性能编码。
