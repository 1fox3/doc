# 静态内部类和普通内部类区别

## 核心区别

| 维度 | 普通内部类 | 静态内部类 |
|------|------------|------------|
| 外部类引用 | 持有外部类实例引用 | 不持有外部类实例引用 |
| 访问外部类成员 | 可直接访问外部类实例成员和静态成员 | 只能直接访问外部类静态成员 |
| 实例化方式 | 需要先创建外部类实例 | 不依赖外部类实例，可直接实例化 |
| 典型用途 | 与外部类实例强关联的逻辑 | Builder、静态内部类单例、枚举式分组 |

## 注意点

- 非静态内部类若被长生命周期对象引用，会导致外部类实例无法被 GC 释放，造成内存泄漏。
