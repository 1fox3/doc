# 重写和重载

## 对比

| 维度 | 重载 Overload | 重写 Override |
|------|--------------|--------------|
| 发生范围 | 同一个类中 | 父子类之间 |
| 参数列表 | 必须不同（个数、类型或顺序） | 必须相同 |
| 返回类型 | 可修改 | 必须相同或为协变返回（子类型） |
| 异常 | 可修改 | 抛出的受检异常不能比父类更宽 |
| 访问修饰符 | 可修改 | 不能比父类更严格（可以放宽） |
| 发生阶段 | 编译期静态分派 | 运行期动态分派 |

## 示例

```java
class Animal {
    void speak() {
        System.out.println("animal");
    }
}

class Dog extends Animal {
    @Override
    void speak() {           // 重写：方法名和参数列表相同
        System.out.println("dog");
    }

    void speak(String prefix) { // 重载：参数列表不同
        System.out.println(prefix + " dog");
    }
}
```
