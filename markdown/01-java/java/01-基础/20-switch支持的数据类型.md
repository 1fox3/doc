## 详细说明

### 支持类型

- 传统 switch 支持 byte、short、char、int 及其包装类型、enum、String。
- 不支持 long、float、double、boolean。
- 新版本 Java 引入 switch expression 和模式匹配能力，表达能力更强。

### 注意点

- String switch 底层会结合 hashCode 和 equals 判断，不能只理解成整数跳转。
