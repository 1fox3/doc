## 重写和重载

### 对比

#### 发生范围

- 重载-同一个类

- 重写-子类

#### 参数列表

- 重载-必须修改

- 重写-一定不能修改

#### 返回类型

- 重载-可修改

- 重写-子类方法返回值类型应比父类方法返回值类型更小或相等

#### 异常

- 重载-可修改

- 重写-子类方法声明抛出的异常应比父类方法声明抛出的异常更小或相等

#### 访问修饰符

- 重载-可修改

- 重写-一定不能做更严格的显示(可以降低限制)

#### 发生阶段

- 重载-编译器

- 重写-运行期
