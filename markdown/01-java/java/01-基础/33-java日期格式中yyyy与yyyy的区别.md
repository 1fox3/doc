## 详细说明

### 核心区别

- `yyyy` 表示自然年 year-of-era。
- `YYYY` 表示 week-based-year，即周所属的年份。
- 跨年周附近使用 `YYYY` 可能把日期格式化到相邻年份，导致线上日期分区或报表错误。

### 实践建议

- 业务日期格式化通常使用 `yyyy-MM-dd`，不要误用 `YYYY-MM-dd`。
