## 详细说明

### 常见方式

- 使用 `new` 调用构造方法。
- 使用反射 `Constructor#newInstance`。
- 使用 clone，但需要实现 `Cloneable` 且语义复杂。
- 通过反序列化恢复对象。
- 通过工厂方法、Builder、Spring 容器创建对象。

### 实践建议

- 业务代码优先使用构造器、静态工厂或 Builder；框架层才大量使用反射和容器创建。
