## 详细说明

### 类型转换

- `s1 += 1` 隐含强制类型转换，等价于 `s1 = (short)(s1 + 1)`。
- `s1 = s1 + 1` 中 `s1 + 1` 会先提升为 int，因此直接赋值给 short 会编译失败。

### 面试重点

- byte、short、char 参与算术运算时通常会提升为 int，这是 Java 数值提升规则的体现。
