## 详细说明

### 核心规则

- 可变长参数本质上是数组，方法内部按数组处理。
- 一个方法最多只能有一个可变长参数，并且必须放在参数列表最后。
- 重载时同时存在数组、固定参数和可变参数可能造成调用歧义。

### 实践建议

- 公共 API 使用可变参数要注意 null、空数组和重载歧义。
