## 详细说明

### 三类位移

- `<<`：左移，低位补 0，相当于在不溢出的情况下乘以 2 的幂。
- `>>`：有符号右移，高位补符号位。
- `>>>`：无符号右移，高位补 0。

### 注意点

- int 位移只取低 5 位，因此移动 32 位等价于移动 0 位；long 位移只取低 6 位。
- 位运算常用于状态标记、权限位、哈希扰动、网络协议和高性能编码。
