# 跳出多层循环

## 常见方式

| 方式 | 说明 |
|------|------|
| `break label` | 在外层循环前定义标签，`break 标签名` 直接跳出该层循环 |
| `return` | 将循环逻辑抽成独立方法，通过 `return` 提前结束 |
| 状态变量 | 用标志位控制外层循环退出，可读性较差，不推荐 |

## 示例

```java
outer:
for (int i = 0; i < 3; i++) {
    for (int j = 0; j < 3; j++) {
        if (i == 1 && j == 1) {
            break outer; // 跳出 outer 标记的外层循环
        }
        System.out.println("i=" + i + ", j=" + j);
    }
}
```

## 注意点

- 业务代码中不要滥用 `label`，复杂嵌套循环更建议拆成独立方法以提升可读性。
