## JDK8提供的内置的函数式接口

### Consumer<T>

#### T最为输入，没有返回值

### Supplier<T>

#### 没有输入，返回T

### Predicate<T>

#### T作为输入，返回boolean

### Function<T, R>

#### T作为输入，R作为输出

### BinaryOperator<T>

#### 两个T作为输入，放回一个T作为输入
