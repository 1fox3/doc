## 接口和抽象类区别

### 共同点

#### 都不能被实例化

#### 都可以包含抽象方法

#### 都可以有默认实现

### 不同点

#### 一个是类，一个时接口

#### 类单继承，接口多继承

#### 接口没有构造方法，而抽象类有

#### 接口中的变量必须是public static final修饰的，是常量且不能被自己和外部修改。抽象类的成员变量默认default，可以在子类被重新定义，也可被重新赋值
