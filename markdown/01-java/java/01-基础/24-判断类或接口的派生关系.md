# 判断类或接口的派生关系

## 常用方式

| 方式 | 说明 | 示例 |
|------|------|------|
| `instanceof` | 判断对象是否是某类型的实例 | `obj instanceof Parent` |
| `Class#isAssignableFrom` | 判断一个 Class 是否可由另一个 Class 赋值（反射场景常用） | `Parent.class.isAssignableFrom(Child.class)` → `true` |
| `Class#isInstance` | 反射版本的 `instanceof` | `Parent.class.isInstance(obj)` |
