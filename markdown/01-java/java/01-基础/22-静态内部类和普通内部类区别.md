## 详细说明

### 核心区别

- 普通内部类持有外部类实例引用，可以访问外部类实例成员。
- 静态内部类不持有外部类实例引用，只能直接访问外部类静态成员。
- 创建普通内部类需要外部类实例，创建静态内部类不需要。

### 典型应用

- 静态内部类常用于 Builder、枚举式分组、静态内部类单例模式。
- 非静态内部类如果被长生命周期对象引用，可能导致外部类实例无法释放。
