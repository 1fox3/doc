## 详细说明

### 核心区别

- Java 运行在 JVM 上，强调跨平台、自动内存管理和安全边界；C/C++ 更接近底层，允许直接操作内存，性能和控制力更强但风险也更高。
- Java 不支持多继承类，但支持接口多继承；C++ 支持类多继承，因此也需要处理菱形继承等复杂问题。
- Java 没有运算符重载和显式指针运算，降低了语言复杂度和内存错误概率。
