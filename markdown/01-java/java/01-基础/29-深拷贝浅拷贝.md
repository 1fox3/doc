## 详细说明

### 核心区别

- 浅拷贝只复制对象本身，引用字段仍指向原来的对象。
- 深拷贝会递归复制引用对象，新旧对象之间不共享可变内部状态。

### 实现方式

- 手写拷贝构造或转换方法，最清晰可控。
- 序列化再反序列化可实现深拷贝，但性能和兼容性一般。
- 使用 MapStruct、BeanUtils 等工具时要明确它们默认多为浅层复制。
