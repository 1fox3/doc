## 详细说明

### 基本规则

- 标准入口是 `public static void main(String[] args)`。
- `main` 可以重载，但 JVM 启动时只识别标准签名。
- `main` 是静态方法，可以被继承但不存在运行期多态重写意义。

### 实践

- Spring Boot 的 main 方法主要负责调用 `SpringApplication.run` 启动应用上下文。
