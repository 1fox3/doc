## StampedLock

### 读写锁

### 不可重入，且不支持条件变量Condition

### 基于CLH锁独立实现的

### 提供三种读写控制模式，且允许切换

#### 写锁

- 独占锁，一把锁只能被一个线程获得

#### 读锁（悲观读）

- 没有线程获取写锁的情况下，多个线程可以同时持有读锁，如果已有线程持有读锁，则其他线程获取读锁会被阻塞

#### 乐观读

- 允许多个线程获取读锁，同时允许一个写线程获取写锁
