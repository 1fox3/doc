## 详细说明

### 常见方式

- 管道、消息队列、共享内存、信号量、Socket、文件、RPC、HTTP、消息中间件。
- 后端分布式系统中更常见的是 HTTP/RPC、消息队列、数据库和缓存作为进程间协作方式。

### 面试重点

- 线程间通信共享同一进程内存，进程间通信需要借助操作系统或网络机制。
