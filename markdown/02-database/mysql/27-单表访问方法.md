## 单表访问方法

### 访问方法

#### const

- 通过主键或唯一二级索引列来定位一条记录

#### ref

- 搜索条件为二级索引列与常数进行等值比较，形成的扫描区为单点扫描区间

#### ref_or_null

- ref_or_null只比ref访问多扫描了一些值为NULL的二级索引记录

#### range

- 使用索引执行查询时，对应得扫描区间为若干个单点扫描区间或者范围扫描区间

#### index

- 扫描全部二级索引

#### all

- 全表扫描

### 索引合并

#### Insersection索引合并

- 两个索引取交集

#### Union索引合并

- 两个索引取并集

#### Sort-Union索引合并

- 两个索引取并集并排序
