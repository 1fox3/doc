## 字符集和比较规则

### 字符集

#### 定义

- 字符串与二进制数据的映射关系

#### 查看字符集

- SHOW CHARSET

#### 特殊字符集

- utf8
  - 1-3个字节

- utf8mb4
  - 1-4个字节

### 比较规则

#### 查看

- SHOW COLLECTION

#### 规则

- -ai
  - 不区分重音

- -as
  - 区分重音

- -ci
  - 不大小写

- -cs
  - 区分大小写

- bin
  - 二进制

### 配置级别

#### 服务器级别

- character_set_server

- collection_server

#### 数据库级别

- CHARACTER SET

- COLLATE

#### 表级别

- CHARACTER SET

- COLLATE

#### 列级别

- CHARACTER SET

- COLLATE

### 客户端和服务器通信字符转换

#### character_set_client

- 服务器认为请求是按照该系统变量指定的字符集进行编码的

#### character_set_connection

- 服务器在处理请求时，会把请求字节序列从character_set_client转换为character_set_connection

#### character_set_results

- 服务器器采用该系统变量制定的字符集对返回给客户端的字符串进行编码
