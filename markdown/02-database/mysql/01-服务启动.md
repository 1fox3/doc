## 服务启动

### 服务启动命令

#### mysqld

- 运行可直接启动一个Mysql服务器程序

#### mysqld_safe

- 启动脚本，可间接调用mysqld，并持续监控服务器的运行状态
  - 服务器出错时，可帮助重启服务器
  - 可记录错误日志

#### mysql.server

- 启动脚本，间接调用mysqld_safe
  - start
  - stop

#### mysqld_multi

- 启动过个服务器实例

### 存储引擎(show engines)

#### ARCHIVE

- 用于数据存档，记录插入后不能再被删除

#### BLACKHOLE

- 丢弃写操作，仅记录操作日志，读操作返回空字符串

#### CSV

- 在存储数据时以逗号分割各个数据项

#### FEDERATED

- 用来访问远程表

#### InnoDB

- 支持事务，行级锁，外键

#### MEMORY

- 数据指存储在内存，不存储在磁盘，多用于临时表

#### MERGE

- 用来管理多个MyISAM表构成的表集合

#### MyISAM

- 主要的非事务处理引擎

#### NDB

- MySQL集群专用存储引擎
