# MongoDB

MongoDB 文档按技术主题拆分。内容以 MongoDB 6.x/7.x/8.x 的通用机制为主，重点覆盖文档模型、索引、事务一致性、复制集、分片集群和运维排查。

## 学习主线

- 基础模型：BSON、数据库、集合、文档、特殊数据库、GridFS、存储引擎。
- 数据建模：围绕访问模式设计文档结构，权衡嵌入、引用、冗余和一致性成本。
- 查询与索引：理解复合索引、多键索引、TTL、唯一索引、覆盖查询和执行计划。
- 一致性：掌握事务、WriteConcern、ReadConcern、ReadPreference、oplog 和因果一致性。
- 高可用与扩展：理解复制集选举、主从同步、分片键、Chunk、Balancer 和迁移流程。
- 运维实践：关注慢查询、容量规划、备份恢复、安全配置和线上故障排查。
